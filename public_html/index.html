twicli

twicli is a Web browser based Twitter client.

  • No need for installation. All you need is a Web browser.
  • Automatically fetches timeline, and show with cool animation.
  • Use cross domain JavaScript to access Twitter directly (no server needed), so your password is sent to Twitter only.

Maybe convenient when you are in an Internet cafe, doing something else while viewing Twitter timeline.
Works on Firefox, Safari, Opera, IE8 at the moment, as well as Safari on iPhone/iPod touch
Works on IE7 too, but some part aren't displayed properly.

Launch twicli

Usage

Launch

Opens a new window when you click the above link, or run as a bookmarklet.
If you get asked for authorization, check if the origin of authorization is really "twitter.com", then enter your Twitter account/password.

Be careful not to hit API request limi (150 requests per hour) by clicking too much ^^;

How to Use

Customize

User Stylesheet

You can add your own CSS in "user stylesheet" section of the "+" tab. For example;
  • Do not show user icons : .uicon { display: none; }
  • Wider lines : .status { line-height: 1.5; }

Plugins

You can register plugins by setting them at the "Preferences" in the "+" tab.
By default, you have those plugins below;
  • regexp.js : Add a tab that extracts timeline that has specific keywords, id, etc.More detail (Japanese)。
  • outputz.js : Count how many letters you tweet, and send it to Outputz.
  • search.js : Add keyword search by Twitter Search to the "+" tab.
  • search2.js : Add keyword search by twitter検索 to the "+" tab.
  • favotter.js : Indicate latest popular favorized tweets like [fav:3]. Thanks to Favotter
  • followers.js : Make tweets of user you don't follow shown in blue. To use this, you need to fetch the list of followers by clicking the "Renew" button at the "Color followers" section. (Turned off by default)
